# Smart Travel Planner
A Python-based travel planning application that helps users plan trips with transport, food, stay, and budget estimation features.

## Features
- Source to destination trip planning
- Via location support
- Budget estimation
- Transport suggestions
- Food recommendations
- Stay recommendations
- Ticket booking links
- User-friendly interface

## Tech Stack
- Python
- VS Code
- HTML
- APIs / Booking Links

## How It Works
1. Enter:
   - From location
   - To location
   - Via location (optional)

2. The system:
   - Calculates estimated budget
   - Suggests transport options
   - Provides food and stay recommendations
   - Generates ticket booking links

## Future Improvements

- Google Maps integration
- AI-based recommendations
